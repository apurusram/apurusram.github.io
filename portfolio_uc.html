The Daily UI Challenge

The Daily UI Challenge is a 100-day design initiative aimed at improving user interface and visual design skills through daily prompts. Each challenge focuses on a specific UI component or screen, such as a login page, profile card, or e-commerce checkout; encouraging creative thinking and consistency. These exercises serve as an opportunity to explore different styles, strengthen design fundamentals, and build a diverse portfolio of practical, user-centered solutions.
