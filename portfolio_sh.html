Improving the User Experience of a Fuel Station Loyalty App

BRIEF

Overview
The Shell Club app serves as a loyalty platform used by a Shell fuel stations, where customers earn points by scanning a QR code after purchasing fuel. These points can be redeemed for gifts displayed within the app, with each gift requiring a certain number of points.

Problem Statement
Customers experience difficulty scanning the small QR code, leading to delays and frustration. Additionally, there is no immediate feedback in the app to confirm whether points have been successfully added. The homepage contains redundant elements that clutter the interface, making navigation less intuitive.

Objective
The goal is to improve the QR code scanning process by making the QR code more accessible and providing users with a notification that points have been added. Additional objectives include adding information about fuel refills, decluttering the homepage by removing redundant sections and enhancing the user interface for a smoother, more intuitive experience.

Needs and Goals

  • Quickly and easily scan the QR code to earn points.
  • Receive a notification confirming points earned after each scan.
  • Navigate the app without encountering clutter or redundant steps.
  • Access details about the last fuel purchase, including location and time.

Pain Points

  • Easily scan the QR code for points.
  • Get notified about points earned after each scan.
  • Navigate the app without unnecessary clutter or redundant steps.
  • Retrieve information about the last fuel purchase.




current task flow

In the current task flow for scanning the QR code, the user logs in to the app and presents the QR code to the fuel station attendant, who scans it. However, the scan often fails on the first try due to the small size of the QR code.




redesigned task flow


In the redesigned task flow, the user logs in, taps on the virtual card to enlarge the QR code, presents the QR code to the fuel station attendant to scan it, and receives an in-app notification confirming the points have been added.

learnings

- User Feedback
Providing immediate feedback, such as a notification confirming points have been added, improves user satisfaction by reducing uncertainty.

- Usability Enhancements
Small changes, like enlarging the QR code, can make a significant difference in how smoothly a key task is performed.

- Decluttering the Interface
Simplifying the homepage by removing redundant elements helps users focus on important features like redeeming points for gifts.

- Reward System
Highlighting the ability to redeem points for gifts encourages user engagement and adds value to the loyalty experience.
